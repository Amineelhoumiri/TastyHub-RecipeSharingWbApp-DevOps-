# Recipe Sharing Web App – Database (COM619)

**Author:** [Hasnain Abbasi]  
**Role:** Database Designer (Group Project)

## 📘 Description
This repository contains the **database design and schema** for the Recipe Sharing Web Application, created as part of the COM619 DevOps project.

The database supports user profiles, recipes, ingredients, comments, and privacy settings.

## 🗂️ Main Tables
- **users** – Stores user info, preferences, and profile data  
- **recipes** – Holds recipe name, steps, ingredients, and privacy settings  
- **comments** – Connects users and recipes with comment text  
- **likes_favorites** – Tracks which user liked or saved a recipe  
- **ingredients** – Lists ingredients for recipes

## ⚙️ How to Use
- Import the provided `.sql` file into **MySQL** or similar DBMS  
- Connect backend using the same table and field names  
- ER diagram provided for reference
